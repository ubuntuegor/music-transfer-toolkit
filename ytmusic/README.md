# YouTube Music

Follow ytmusicapi setup instructions first: https://ytmusicapi.readthedocs.io/en/stable/setup/index.html

`clear.user.js` is a userscript instead because ytmusicapi was unreliable for clearing the "LIKED" status.

## Warning

Importing is flaky and some songs that were found might not get liked/added properly. I recommend exporting the playlist when the import is finished, and then checking that all video IDs from import results are present in the export.
